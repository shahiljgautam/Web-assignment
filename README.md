# University Campus Portal

## Project Overview

The **University Campus Portal** is a responsive, front-end web project created using **HTML5 and CSS3**. This project represents a modern college website that includes a student resume section, registration form, campus map, and institutional information. It is designed for academic purposes and demonstrates clean UI design, responsive layout techniques, and structured web development practices.

## Objectives

* To design a responsive university website layout
* To demonstrate the use of semantic HTML5 elements
* To apply modern CSS techniques such as Grid, Flexbox, and CSS variables
* To create a student-focused resume and registration interface
* To integrate external resources like icons and Google Maps

## Technologies Used

* **HTML5** – Page structure and content
* **CSS3** – Styling, layout, responsiveness
* **Font Awesome** – Social media and UI icons
* **Google Maps Embed** – Campus location display

## Key Features

### 1. Navigation Bar

* Fixed navigation bar for easy access
* Responsive hamburger menu for mobile devices
* Smooth scrolling between sections

### 2. Student Resume Section

* Two-column grid layout
* Left column: student photo, personal details, social links
* Right column: technical skills with animated progress bars
* Bottom grid displaying four selected projects

### 3. Skill Analytics

* Skill levels represented using animated progress bars
* Visual indication of proficiency (Advanced / Intermediate)
* Uses CSS transitions for smooth loading animation

### 4. Student Registration Form

* Modern, clean UI with grid-based layout
* HTML5 form validation (required fields, email, phone pattern)
* File upload support for documents
* Visual feedback for valid and invalid inputs

### 5. Campus Map

* Embedded Google Map showing Kathmandu Business Campus
* Responsive iframe container

### 6. Footer Section

* College contact information
* Social media links with hover effects
* Copyright information

## Responsive Design

* CSS Grid and Flexbox used for layout
* Media queries ensure compatibility across:

  * Desktop
  * Tablet
  * Mobile devices
* Layout automatically adapts to screen size

## Design Highlights

* CSS variables for consistent color usage
* Hover and focus effects for better user experience
* Rounded components and smooth transitions
* College branding colors applied throughout the site
